For running the Q3.py file, simply run the code on any ide or editor.
First it will print out the train and test errors for 3.1. Then it will show a plot for 3.2 for train and test errors over the l2 norm. 
Then it will print normalized train and test errors for each of the alphas over the 10 iterations as part of 3.3. Then it will plot two more graphs for parts 3.4 and 3.5.

For running the Q4.py file just simply run the code on any ide or editor. 
First it will show the graph of Q-4.1) where it will show 6 lines and as soon as you close that window it will start executing 
the Q-4.2) part of the code which will again plot 0-1 loss on different eta.
